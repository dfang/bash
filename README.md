### Install homesick  
`sudo gem install homesick`

### Clone castle
`homesick clone dfang/bash`  

### Link castle  
`homesick link bash`  

### Update castle
`homesick pull bash`  

### Install some homebrew formula first(check docs/.BREWS_AND_GEMS in castle bash)
`brew install fasd fzf tree ssh-copy-id`
